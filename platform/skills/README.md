# Skills — project-specific how-to library

Reusable, advisory guides the agents can `read_file` while working. They encode SmartCart conventions so
agents produce code that matches the existing codebase. They are documentation, not executable code.

- [`write-nest-module.md`](write-nest-module.md) — add a NestJS feature module to `apps/api`
- [`write-rn-component.md`](write-rn-component.md) — add an atomic-design component to the Expo frontend
- [`prisma-migration.md`](prisma-migration.md) — evolve the Prisma schema safely
- [`run-tests.md`](run-tests.md) — the test/quality-gate commands the platform runs
- [`open-pr.md`](open-pr.md) — branch + PR conventions used by `/release-feature`
- [`owasp-checklist.md`](owasp-checklist.md) — the security checklist the validation agent enforces
